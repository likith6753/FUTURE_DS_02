# Customer Churn & Retention Analysis

## Project Overview

This project analyzes customer churn and retention patterns using subscription data to identify why customers leave and what drives long-term engagement. The analysis focuses on customer segmentation, churn trends, cohort behavior, and customer lifetime to generate actionable business insights that help improve retention and reduce revenue loss.

## Objectives

* Clean and prepare customer subscription data
* Analyze churn rates and retention trends
* Perform customer cohort analysis by signup period, plan, or region
* Evaluate customer lifetime patterns
* Identify key factors influencing customer churn
* Provide data-driven recommendations to improve customer retention

## Tools Used

* Microsoft Excel – Data cleaning and exploratory analysis
* Power BI – Interactive dashboard and data visualization
* Python (Optional) – Advanced analysis and churn metrics

Key Insights

* Identified customer segments with the highest churn risk
* Analyzed retention trends across different customer cohorts
* Measured customer lifetime and engagement patterns
* Highlighted factors contributing to customer churn
* Developed actionable strategies to improve retention and customer loyalty

Deliverables

* Interactive Power BI Retention Dashboard
* Customer Churn Analysis Report
* Business Recommendations for Retention Improvement

Skills Demonstrated

* Customer Churn Analysis
* Cohort Analysis
* Customer Lifetime Value (CLV) Analysis
* Data Cleaning and Transformation
* Data Visualization
* Business Intelligence
* Data-Driven Decision Making

Business Value

This project demonstrates how data analytics can support SaaS and subscription-based businesses by identifying churn risks, improving customer retention, and enabling informed business decisions that drive sustainable growth.
